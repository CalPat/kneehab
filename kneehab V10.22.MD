# KneeHab V10.22 — Cossack R0: Overhead Band

**Date:** 2026-03-31
**Change type:** Minor

## Summary

Adds R0 (overhead band) to the Cossack Squat regression ladder. An elastic band looped from a ceiling anchor reduces bodyweight load at depth, enabling full hip and knee ROM when compressive load is the limiting factor — as distinct from R1 (door frame/TRX), which assists balance/pattern but does not reduce load.

## Changes

### Cossack Squat — regression ladder (BLOCKS JS + src-knee HTML)
- **R0 added:** Overhead band — loop band from ceiling anchor; reduces bodyweight at depth, enables full ROM. Use when load is the limiting factor.
- **R1 clarified:** Assisted (door frame / TRX) — use when pattern/balance is the limiting factor.
- Start cue updated: "Start at R0 if load is the limiting factor, R1 if pattern/balance."

## Rationale

Overhead band provides elastic counterforce that is greatest at bottom of movement (maximum band stretch = maximum offload), directly where compressive load and ROM demand peak. This is more targeted than TRX/door frame for users limited by joint load rather than balance.
