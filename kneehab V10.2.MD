# KneeHab V10.2 — Asymmetry Rule + Frontal Plane Rebuild

**Date:** 2026-03-30
**Change type:** Medium (12 changes across 4 blocks)

## Summary

Introduces the Asymmetry Rule protocol and rebuilds the Load day around a confirmed frontal plane deficit. Load day trimmed by removing Recovery-day duplicates (Supine Hip ER, RKC Plank, Face Pull) and reducing Couch Stretch. Skater Squat removed as too advanced for current frontal plane capacity. Curtsy Squat and Cossack Squat added as direct glute med training in adducted/lateral loading patterns. Calf Raises moved to Optional pool. Net session time is approximately neutral despite adding two exercises.

## Changes

### Asymmetry Rule (new protocol)
Added to Load Day briefing: weaker side first on all alternating exercises, extra set to weaker side.

### Mobility Block (Load Day)
- **Couch Stretch:** 2 × 30s → 1 × 30s on Load day. Full volume retained on Recovery day.

### Spinal & Shoulder Block (Load Day)
- **Face Pull:** removed from Load day. Recovery day unchanged.

### Core Block (Load Day)
- **RKC Plank:** removed from Load day. Recovery day unchanged.

### Glute Block (Load Day)
- **Supine Hip ER with Block:** removed from Load day. Recovery day unchanged.
- **Hip Abduction:** rx updated to `3 × 12 weak / 2 × 12 strong · 2s hold top`. Asymmetry rule applied.
- **Adductor ISO Hold:** asymmetry rule applied to description.

### Knee & Lower Body Block
- **Slant Board Step Down:** pre-activation cue added — 5s single-leg stance, level pelvis, before weaker-side sets.
- **Skater Squat:** removed. Frontal plane control not yet sufficient for single-leg demand.
- **Curtsy Squat:** added. `3 × 8 weak / 2 × 8 strong · 2s descent`. Bilateral crossed stance, high glute med demand in adducted position. Asymmetry rule applies.
- **Cossack Squat:** added. `3 × 5 weak / 2 × 5 strong · 3s descent · 3s ascent`. Lateral movement pattern with full regression ladder (Assisted → Elevated Heel → Partial Depth → Full → Weighted). Start at R1. Asymmetry rule applies.
- **Calf Raise ×2:** moved to Optional pool.
- Block timing updated: ~17 min → ~20 min.

### Optional Pool
- **Calf Raise ×2:** added. Joins Copenhagen Plank, Hamstring ISO in Flexion, TKE, Single-Leg Wall Sit, Hanging Knee Raise.

## Rationale

Frontal plane deficiency (glute med weakness) is the primary upstream driver of current compensation patterns. Exercises reorganised to address this deficit at every level of demand: isolated (Hip Abduction), bilateral-crossed (Curtsy Squat), and lateral-pattern (Cossack Squat). The Asymmetry Rule ensures the weaker side receives more stimulus each session without adding global volume. Load day trimmed to offset the added knee block time.
